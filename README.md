# ESP32 Environment Sensor 🌡️

A simple IoT project built in Wokwi that monitors temperature and humidity.

## Hardware Used
* **ESP32** DevKit
* **DHT22** Sensor
* **LCD 20x4** (I2C)

## Connectivity
* **DHT22 Data:** GPIO 32
* **LCD SDA:** GPIO 21
* **LCD SCL:** GPIO 22

## How to use
1. Open the `diagram.json` in Wokwi.
2. Copy the code from `main.ino`.
3. Run the simulation!
